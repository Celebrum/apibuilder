# ApiDOM

All following commands should be executed from top level `package.json`.

## Installation

```sh
 $ npm i
 $ npm run bootstrap
```

## Tests

```sh
 $ npm run test
```

## Linting

```sh
 $ npm run lint
```

## TypeScript types checking


```sh
 $ npm run check-types
```

## Building artifacts

```sh
 $ npm run build
```
